# Solar System
A java + LibGDX real-time 3D solar system simulation.


# Helpful commands
You can build all available distributions with `./gradlew build`. Alternatively, you can build only the desired one with one of the following commands:
- `LinuxArm32`
- `LinuxArm64`
- `LinuxX64`
- `Windows32`
- `Windows64`
- `MacosArm64`
- `MacosX64`
